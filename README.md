
## Langkah setup

**buka terminal dan jalankan perintah berikut**
 - `git clone https://github.com/mardiansyah17/propeti.git`
 - `composer install`
 - `cp .env.example .env`
 - `php artisan key:generate`
 - `php artisan migrate`
   
 **buka terminal baru dan jalankan perintah berikut**
 
 - `npm run dev`
